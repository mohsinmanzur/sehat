# Sehat — Marketing Landing Page

Static HTML/CSS/JS marketing site (no build framework) for Sehat, a medical report digitization app. Deployed to mysehat.app via GitHub Pages. `index.html` is the single real page; `phone.html` is a fragment fetched at runtime into the hero's interactive phone mockup.

## Design Context

Full strategic and visual specs live in [PRODUCT.md](PRODUCT.md) and [DESIGN.md](DESIGN.md) — read both before making design changes.

- **Register:** brand (marketing landing page) · **Platform:** web
- **North Star:** "The Quiet Instrument" — clinical calm, one accent color (Cobalt `#2A5CFF`), pill shapes, ambient-to-lifted shadow on hover only
- **Anti-references:** no stock-photo doctors/stethoscopes, no teal-corporate-healthcare palette, no generic AI-marketing scaffolding (gradient text, floating eyebrows, numbered section markers)
- **Primary goal:** drive Android APK downloads; audience is South Asian patients managing paper lab reports
