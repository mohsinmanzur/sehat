# SehatScan Doctor Portal

React + Vite frontend built from the UX Pilot doctor portal designs.

## Run

```bash
npm install
npm run dev
```

## Pages
- /login
- /verify
- /dashboard
- /access
- /overview
- /reports
- /reports/:id
- /sessions
- /settings

## Notes
- Frontend only
- Uses mock data and mocked auth/session flow
- Dark mode is included and text colors are theme-safe
